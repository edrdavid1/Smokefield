Политика конфиденциальность

Приложение получает доступ к камере для сканирования QR-кодов. Данные, полученные через камеру, сохраняются только на стороне пользователя, если это необходимо для работы функционала. Сервис собирает такую информацию, как введённое имя пользователя, результаты сканирования и передаваемые значения (например, “цигареты”). Передача данных на сервер происходит для обновления счётчиков или статистики пользователя. Сервис не передаёт полученные данные третьим лицам без обязательств, предусмотренных законом. Локальное хранилище (localStorage) используется для сохранения имени пользователя и упрощения доступа к функционалу. Пользователи могут в любой момент очистить локальные данные или отказаться от предоставления доступа к камере, но некоторые функции при этом перестанут работать. Обработка данных ведётся только с целью обеспечения корректной работы сервиса и обратной связи с пользователем.
